# Инструкции для устройств

## iPhone и iPad

1. Получите ссылку подписки в личном кабинете.
2. Установите HAPP.
3. Откройте ссылку на iPhone или iPad.
4. Разрешите добавление VPN-конфигурации.
5. Включите подключение в HAPP.

## Android

1. Установите HAPP из официального источника.
2. Откройте ссылку подписки на Android.
3. Если ссылка не открылась автоматически, скопируйте её и добавьте как подписку внутри HAPP.
4. Разрешите создание VPN-подключения.

## Windows

1. Установите HAPP для Windows.
2. Откройте или скопируйте ссылку подписки.
3. Добавьте её в HAPP как подписку.
4. Включите профиль SimpleGate.

## macOS

1. Установите HAPP для macOS.
2. Откройте ссылку подписки или добавьте её вручную.
3. Разрешите системный запрос на сетевую/VPN-конфигурацию.
4. Включите подключение.

## Linux

1. Установите HAPP для своего дистрибутива, если доступен подходящий пакет.
2. Добавьте ссылку подписки.
3. Включите соединение.
4. Если подключение не стартует, проверьте права на сетевые настройки.

## Android TV / Google TV

HAPP на Android TV можно установить через Google Play или APK. При первом запуске приложение может предложить импорт подписки через QR-код или локальную сеть. Если локальная передача не сработала, используйте Web Import.

## Apple TV

Установите HAPP для tvOS. Для передачи подписки удобнее использовать QR-код, Web Import или локальную передачу с телефона. Apple TV считается отдельным устройством в лимите тарифа.
