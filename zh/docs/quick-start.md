# SimpleGate 快速开始

## 1. 打开个人中心

打开 SimpleGate Telegram 机器人：https://t.me/SimpleGateVPN_bot

你可以开启 3 天免费试用，选择套餐，完成付款，并获取个人订阅链接。

## 2. 安装 HAPP

HAPP 是用于添加 SimpleGate 连接的应用。它支持手机、电脑和电视平台。

## 3. 打开订阅链接

开启试用或付款后，在已安装 HAPP 的设备上打开订阅链接。如果系统提示用 HAPP 打开，请确认。

## 4. 开启连接

打开 HAPP 并开启连接。系统可能会请求添加 VPN 配置权限，这是正常请求。

## 免费使用

新用户可以先使用 3 天免费试用。拥有有效付费订阅的用户可以通过推荐链接邀请朋友并获得免费使用天数。
