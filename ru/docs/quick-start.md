# Как быстро подключить SimpleGate

## 1. Откройте личный кабинет

Перейдите в Telegram-бот SimpleGate: https://t.me/SimpleGateVPN_bot

В кабинете можно получить 3 дня бесплатного теста, выбрать тариф, оплатить доступ и получить личную ссылку подписки.

## 2. Установите HAPP

HAPP — приложение, через которое добавляется подключение SimpleGate. Оно работает на телефонах, компьютерах и TV-платформах.

## 3. Откройте ссылку подписки

После теста или оплаты откройте ссылку подписки на устройстве, где установлен HAPP. Если система предложит открыть ссылку в HAPP — подтвердите.

## 4. Включите подключение

Откройте HAPP и включите соединение. При первом запуске система может запросить разрешение на добавление VPN-конфигурации.

## Как пользоваться бесплатно

Новые пользователи начинают с 3 дней бесплатного теста. Пользователи с активной платной подпиской могут получить реферальную ссылку и получать бесплатные дни за приглашения.
