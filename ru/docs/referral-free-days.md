# Как пользоваться SimpleGate бесплатно

В SimpleGate есть реферальная программа для пользователей с активной платной подпиской.

## Как работает

1. Откройте личный кабинет.
2. Нажмите «Пригласить друга».
3. Отправьте свою реферальную ссылку.
4. Друг может сначала взять бесплатный тест на 3 дня.
5. После первой оплаты друга вам добавятся бесплатные дни.
6. Если друг потом пригласит своего друга, можно получить ещё дни сверху.

## Условия

- бонус начисляется только после первой оплаты приглашённого пользователя;
- за бесплатный тест бонусные дни не начисляются;
- повторные оплаты одного и того же друга повторный бонус не дают;
- реферальная программа доступна пользователям с активной платной подпиской.
