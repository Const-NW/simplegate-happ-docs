# SimpleGate — приватный VPN с подключением через HAPP

SimpleGate — приватный VPN-сервис для телефона, компьютера и ТВ-приставок. Подключение выполняется через HAPP по личной ссылке подписки.

## Что есть

- 3 дня бесплатного теста для новых пользователей;
- тарифы на 1, 2, 3, 5 и 10 устройств;
- HAPP без сложной ручной настройки;
- iOS, Android, Windows, macOS, Linux, Apple TV, Android TV и Google TV;
- личный кабинет в Telegram;
- запасной вход через VK ID;
- реферальная программа с бесплатными днями.

## Быстрый старт

1. Откройте личный кабинет.
2. Получите тест или выберите тариф.
3. Установите HAPP.
4. Откройте личную ссылку подписки.
5. Включите подключение.

Личный кабинет: https://t.me/SimpleGateVPN_bot  
Поддержка: https://t.me/SimpleGate_bot

## Важно

HAPP — это приложение-клиент. Само приложение HAPP не выдаёт VPN-доступ и не предоставляет серверы. Доступ, ссылку подписки и поддержку предоставляет SimpleGate.
