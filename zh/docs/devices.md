# 设备教程

## iPhone 和 iPad

1. 在个人中心获取订阅链接。
2. 安装 HAPP。
3. 在 iPhone 或 iPad 上打开链接。
4. 按系统提示确认。
5. 在 HAPP 中开启连接。

## Android

1. 从官方来源安装 HAPP。
2. 在 Android 上打开订阅链接。
3. 如果没有自动打开，请复制链接并在 HAPP 中添加订阅。
4. 按系统提示确认。

## Windows

1. 安装 Windows 版 HAPP。
2. 打开或复制订阅链接。
3. 在 HAPP 中添加订阅。
4. 开启 SimpleGate 配置。

## macOS

1. 安装 macOS 版 HAPP。
2. 打开订阅链接或手动添加。
3. 按系统提示确认。
4. 开启连接。

## Linux

1. 如果有适合你发行版的包，安装 Linux 版 HAPP。
2. 添加订阅链接。
3. 开启连接。
4. 如果无法连接，请联系支持。

## Android TV / Google TV

Android TV 版 HAPP 可通过 Google Play 或 APK 安装。首次启动时，可以通过 QR 码、本地方式或 Web Import 添加订阅。

## Apple TV

安装 tvOS 版 HAPP。通常可以通过 QR 码、Web Import 或从手机传输订阅。Apple TV 会计入套餐设备数量。
