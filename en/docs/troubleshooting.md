# Troubleshooting

## Link does not open

Make sure HAPP is installed, open the link on the correct device, or copy it and add it as a subscription.

## Connection does not start

Allow the VPN configuration request and check that your trial or plan is active.

## Russian services

Russian services should normally work while VPN is enabled. If a specific service does not open, contact support and include your device and example website.
