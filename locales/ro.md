# SimpleGate în română

SimpleGate este un serviciu VPN privat pentru telefon, computer și dispozitive TV. Conectarea se adaugă în aplicația HAPP folosind un link personal de abonament.

## Start rapid

1. Deschideți contul personal: https://t.me/SimpleGateVPN_bot
2. Activați testul gratuit de 3 zile sau alegeți un tarif.
3. Instalați HAPP.
4. Deschideți linkul personal de abonament pe dispozitivul dorit.
5. Porniți conexiunea în HAPP.

## Zile gratuite

Utilizatorii cu abonament plătit activ pot invita prieteni printr-un link de recomandare. După prima plată a utilizatorului invitat, se adaugă zile gratuite. Pentru testul gratuit nu se acordă bonus.
