# 如何免费使用 SimpleGate

SimpleGate 为拥有有效付费订阅的用户提供推荐功能。

## 工作方式

1. 打开个人中心。
2. 点击“邀请朋友”。
3. 发送你的推荐链接。
4. 朋友可以先使用 3 天免费试用。
5. 朋友首次付费后，你的账号会获得免费天数。
6. 如果朋友之后又邀请其他用户，你也可以获得额外天数。

## 条件

- 只有在被邀请用户首次付费后才发放奖励；
- 免费试用不会产生奖励；
- 同一个被邀请用户重复付费不会再次产生同样奖励；
- 推荐功能面向拥有有效付费订阅的用户。
