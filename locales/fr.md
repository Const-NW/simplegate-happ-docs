# SimpleGate en français

SimpleGate est un service VPN privé pour téléphone, ordinateur et appareils TV. La connexion est ajoutée dans HAPP avec un lien personnel d’abonnement.

## Démarrage rapide

1. Ouvrez le portail personnel : https://t.me/SimpleGateVPN_bot
2. Activez l’essai gratuit de 3 jours ou choisissez une offre.
3. Installez HAPP.
4. Ouvrez votre lien personnel d’abonnement sur l’appareil souhaité.
5. Activez la connexion dans HAPP.

## Jours gratuits

Les utilisateurs avec un abonnement payant actif peuvent inviter des amis via un lien de parrainage. Après le premier paiement de l’utilisateur invité, des jours gratuits sont ajoutés. L’essai gratuit ne donne pas de bonus.
