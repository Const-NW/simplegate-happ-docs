# Device guides

## iPhone and iPad

1. Get the subscription link in the account portal.
2. Install HAPP.
3. Open the link on iPhone or iPad.
4. Allow VPN configuration.
5. Turn on the connection in HAPP.

## Android

1. Install HAPP from an official source.
2. Open the subscription link on Android.
3. If it does not open automatically, copy it and add it as a subscription in HAPP.
4. Allow VPN connection creation.

## Windows

1. Install HAPP for Windows.
2. Open or copy the subscription link.
3. Add it in HAPP as a subscription.
4. Turn on the SimpleGate profile.

## macOS

1. Install HAPP for macOS.
2. Open the subscription link or add it manually.
3. Allow the system network/VPN configuration request.
4. Turn on the connection.

## Linux

1. Install HAPP for your distribution if a suitable package is available.
2. Add the subscription link.
3. Turn on the connection.
4. If it does not start, check network permissions.

## Android TV / Google TV

HAPP on Android TV can be installed via Google Play or APK. On first launch, it may offer subscription import via QR code or local network. If local transfer fails, use Web Import.

## Apple TV

Install HAPP for tvOS. QR code, Web Import or local transfer from a phone are usually the easiest methods. Apple TV counts as a separate device in your plan limit.
