# FAQ SimpleGate

## Что такое SimpleGate?

Приватный VPN-сервис для телефона, компьютера и TV. Подключение выполняется через HAPP.

## Что такое HAPP?

Клиент для proxy/VPN-конфигураций и подписок. HAPP сам не предоставляет серверы.

## Есть ли тест?

Да, 3 дня бесплатно для новых пользователей.

## Можно ли пользоваться бесплатно?

Да, через реферальную программу после активации платной подписки.

## Когда начисляются дни?

После первой оплаты приглашённого пользователя. За тест бонус не начисляется.
