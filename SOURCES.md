# Source notes

Facts used from HAPP public documentation:

- HAPP is a cross-platform app for proxy servers built on Xray.
- HAPP is available for Windows, macOS, Linux, iOS, Android and TV platforms.
- HAPP supports VLESS, VMess, Trojan, Shadowsocks and Socks.
- HAPP itself does not include servers and does not provide VPN services.
- HAPP supports adding configurations/subscriptions through URL, QR code, deep link and clipboard.
- HAPP supports encrypted subscriptions starting with `happ://crypto...`.
- Android TV supports subscription transfer through QR/local network and Web Import.

Official HAPP site: https://www.happ.su/main
