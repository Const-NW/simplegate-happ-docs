# SimpleGate — 通过 HAPP 使用的私人 VPN

SimpleGate 是面向手机、电脑和电视设备的私人 VPN 服务。用户在 HAPP 中添加个人订阅链接即可连接。

## 提供内容

- 新用户 3 天免费试用；
- 支持 1、2、3、5、10 台设备的套餐；
- 通过 HAPP 连接，不需要复杂手动配置；
- 支持 iOS、Android、Windows、macOS、Linux、Apple TV、Android TV 和 Google TV；
- Telegram 个人中心；
- VK ID 备用入口；
- 推荐奖励，可获得免费使用天数。

## 快速开始

1. 打开个人中心。
2. 开启试用或选择套餐。
3. 安装 HAPP。
4. 打开个人订阅链接。
5. 在 HAPP 中开启连接。

## 重要说明

HAPP 是客户端应用。HAPP 本身不提供 VPN 服务器，也不直接出售 VPN 服务。SimpleGate 提供访问权限、订阅链接和支持。
